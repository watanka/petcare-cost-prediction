### API Endpoint
- /predict `POST`: request: 관련 정보(입력 또는 DB로부터 수집) 기반 양육비 예측 결과 요청
- /analyze `GET` : 분석 페이지(streamlit)로 연결 -> 관리자 쪽에서 분석 결과 확인
- /experiment `GET` : mlflow 실험관리 -> 관리자 쪽에서 실험 결과 확인
